# Centro-Multimedia
Proyecto final para la asignatura de Fundamentos de Sistemas Embebidos. 
Aplicación embebida de centro multimedia para Raspberry Pi 4, usando Chromium y Xorg.

### Desarrollado por:
Galván Zúñiga Adrián Ricardo  
Lozano Bustamante Miguel Alejandro  

Contiene ocho archivos:  
  Los cinco de _home_ se colocan en el directorio homónimo de la Raspberry.  
  Los tres de _config_ se colocan en los directorios indicados en el documento tipo manual.
